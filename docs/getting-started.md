开始使用
========

如何安装
--------

```bash
git clone https://github.com/gengen1988/static-html-builder.git
cd static-html-builder
npm install
```

文件结构说明
--------

目录名           | 介绍
---------------- | --------------
app              | 项目目录
bower_components | bower 存放路径
dist             | 编译后文件的存放路径
tasks            | grunt 任务配置


如何引入第三方依赖
-----

```bash
bower install <包名>
```
